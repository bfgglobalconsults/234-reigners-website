# Fonts Directory

## Font Files

The Nuosu SIL font has been successfully installed and configured.

### Nuosu SIL

- **File:** `NuosuSIL-Regular.ttf`
- **Location:** `src/fonts/NuosuSIL-Regular.ttf` (for Next.js localFont)
- **Also available in:** `public/fonts/NuosuSIL-Regular.ttf` (for direct CSS usage)
- **Source:** [SIL International - Nuosu SIL](https://software.sil.org/nuosu/)
- **License:** SIL Open Font License

The font is configured in the application and is used for all major headers (h1-h6).

### Font Configuration

- **Headers (h1-h6):** Nuosu SIL
- **Body text and paragraphs:** Nunito Sans

Both fonts are loaded via the layout file and applied globally through Tailwind CSS.
